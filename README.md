# Sistema de Cotrole de Invetário

## Trabalho da Matéria de Estrutura de Dados Lineares em C++

* Trabalho voltado para a compreenção do uso da tabela hash desenvolvido na linguagem C++
* Trata-se de um sistema de controle de inventário de um laboratório institucional, como empréstimo de:
  * Impressoras 3D,
  * Kits de robótica,
  * Componentes eletrônicos, máquinas de corte,
  * Sensores e instrumentos de bancada.
